Description
This repository contains Playwright scripts for testing the SwiftTranslator website (https://www.swifttranslator.com/), along with an Excel file of functional test cases.

How to Run
1. Clone the repository: https://github.com/Apurva-Herath/IT3040_Assignment1_IT23759220.git
2. Navigate to the project folder: IT3040_Assignment1_IT23759220
3. Install dependencies: npm install
4. Install Playwright browsers: npx playwright install
5. Run all tests : npx playwright test
6. Run a specific test file : npx playwright test tests/pos_fun_0001_spec.js
7. Generate a test report : npx playwright show-report
